# 50 种相机设计 Tips（第三人称动态相机）

# 1 在没有特殊的需求时，优先使用固定视角的相机
固定视角不丢人，没必要一定用自由视角
# 2 关卡与相机行为应该匹配
想象一下超级玛丽如果是第一人称相机，这个游戏该有多糟糕
# 3 相机状态
首先，自由视角相机永远不要原地旋转，你需要将相机想象为围绕 Avatar 的**轨道**
相机的状态一般有七个参数 Yaw、Pitch、Roll（一般而言不会动 Roll，除非是反重力视角等特殊需求）、距离、横向和垂直的偏移、FOV
